# Ashveil RPG

Ashveil is an atmospheric, tactical, browser-based rogue-lite RPG centered around the dark siege of Ashveil Castle. Players control a hero climbing through dangerous floors—the Gatehouse, the Great Hall, and the final Throne Room—facing corrupted guards, dark cultists, and eventually the Hollow King himself.

This project is built using modern **React**, **TypeScript**, **Vite**, and **Tailwind CSS**, with fully synthetic procedural music and sound effects synthesized entirely in real-time using **Tone.js**.

---

## 🏰 Game World & Setting

Ashveil Castle was once the glowing beacon of the northern borders. Three days ago, the beacon went silent. You have arrived to find a ruin reclaimed by dark forces. The layout changes constantly, distorted by the grief of the Hollow King.

### 🤺 Chosen Heroes

*   **The Knight (The Sworn Blade):** Armed with heavy iron armor and a sworn shield. Uses *Shield Bash* to stagger enemies and *Second Wind* to endure the long siege.
*   **The Mage (The Ashbound Scholar):** Harnesses volatile arcane elements. Casts destructive *Fireballs* and molds defensive *Frost Wards* to shield against dark strikes.
*   **The Rogue (The Silent Blade):** Blends into the castle's shadows. Delivers lethal *Backstabs* to unsuspecting targets and escapes critical situations with *Smoke Bombs*.

---

## 🛠️ Features

*   **Procedurally Generated Maps:** Each floor is procedurally carved with unique room structures, hidden items, locked chests, and portals.
*   **Widescreen Responsive UI:** A beautifully polished desktop dashboard layout featuring rich status grids, active skill hotbars, interactive inventories, and real-time adventure chronicles.
*   **Interactive NPC Dialogue:** Encounter various lost souls within the castle, engaging in branch-based storytelling to uncover the tragedy of King Aldric.
*   **Dynamic Sound Engine:** Real-time audio synthesis powered by **Tone.js** that shifts key, tempo, and atmosphere depending on your floor level and active combat state.
*   **Procedural Embers & Weather:** Falling ash particles and animated visual states that breathe life into the dark fantasy atmosphere.

## 🚀 Getting Started

To run the application locally, follow these steps:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000`.

### Building for Production

To compile the application:
```bash
npm run build
```
